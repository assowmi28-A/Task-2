# 📊 Data Visualization and Storytelling

## 📌 Project Overview

This project was completed as part of the **Data Analyst Internship – Task 2: Data Visualization and Storytelling**.

The objective of this project is to create meaningful visualizations that communicate business insights effectively using Power BI or Tableau. The focus is on transforming raw sales data into an engaging visual story that supports better decision-making.

---

## 🎯 Objective

- Create clear and impactful visualizations
- Present business insights through storytelling
- Use appropriate chart types for different analyses
- Highlight key trends and patterns
- Build an easy-to-understand visual report

---

## 🛠️ Tools Used

- Power BI / Tableau
- Microsoft Excel / CSV Dataset
- GitHub

---

## 📂 Dataset

**Dataset Used:** Superstore Sales Dataset

The dataset includes:
- Order Date
- Sales
- Profit
- Quantity
- Category
- Sub-Category
- Region
- Customer Segment
- Discount

---

## 📈 Visualizations Created

- 📊 Sales by Category
- 📈 Monthly Sales Trend
- 💰 Profit by Region
- 🥧 Sales Distribution by Segment
- 🏆 Top 10 Products by Sales
- 📉 Profit vs Sales Comparison

---

## 📖 Business Insights

- Technology category generated the highest sales.
- West region contributed the highest overall revenue.
- Some products had high sales but low profitability.
- Monthly trend analysis identified seasonal sales patterns.
- Consumer segment accounted for the largest share of sales.
